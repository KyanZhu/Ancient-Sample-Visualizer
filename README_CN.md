# 基因组数据可视化工具

这个项目是一个基于网页的可视化工具，用于展示 Harvard Dataverse 整理的样本信息。

## 数据来源

数据来源于 [Harvard Dataverse](https://dataverse.harvard.edu/dataset.xhtml?persistentId=doi:10.7910/DVN/FFIDCW)。

## 功能特点
- 交互式数据可视化
- 支持数据筛选和查询

## 使用方法

1. 打开index.html：
   - 打开采样信息文件
   - 调整可视化参数

2. 数据格式要求：
   - 支持 XLSX, CSV 格式
   - 最少需要的数据列：经纬度信息 [Long.] [Lat.]
